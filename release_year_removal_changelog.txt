================================================================================
  CHANGE 1: RELEASE_YEAR REMOVAL (2026-04-30)
================================================================================

  PROBLEM: release_year was the single most influential predictor (11.14%
  importance, Pearson correlation -0.370), but it created survivorship bias
  and temporal leakage. Older games appeared more "successful" simply because
  they had more years to accumulate estimated owners.

  DECISION: Remove entirely and retrain.

  IMPACT: 59 → 58 features, Accuracy 80.55% → 78.46%, F1 66.24% → 62.95%.
  The drop was acceptable — the model removed its strongest but biased predictor.


================================================================================
  CHANGE 2: has_reviews REMOVAL (2026-05-06)
================================================================================

  PROBLEM: has_reviews was a binary flag derived from num_reviews_total != -1.
  While originally defended as a "data quality flag," deeper analysis revealed:
    - Strong class correlation: Flop 51%, Niche Hit 82%, Successful 97%
    - It is fundamentally derived from post-launch review data
    - We already removed num_reviews_total as leakage — keeping its binary
      derivative was inconsistent

  DECISION: Remove has_reviews from the model entirely.


================================================================================
  CHANGE 3: 5 NEW FEATURES ADDED (2026-05-06)
================================================================================

  To compensate for has_reviews removal and improve model quality, an
  exhaustive sweep of 17 candidate features was conducted. Five features
  passed all safety checks:

  Feature            Source              Importance  Leakage Check
  ----------------------------------------------------------------
  tag_count          Steam tags dict     8.69% (#1)  corr=0.264 CLEAN
  screenshot_count   screenshots list    5.71% (#5)  corr=0.147 CLEAN
  has_website        website URL field   2.19% (#10) corr=0.206 CLEAN
  has_workshop       categories field    0.47% (#55) corr=0.141 CLEAN
  required_age       required_age col    0.62% (#50) corr=0.163 CLEAN

  All 5 features are:
  - 100% available before game launch
  - Developer-controllable decisions
  - Free from data leakage (all correlations with target < 0.3)
  - Independent from existing features (no pairwise correlation > 0.7)

  REJECTED CANDIDATES (insufficient impact):
  - has_vr (+0.32pp), has_cloud_save (+0.28pp), has_cross_platform (+0.27pp),
    has_support_email (+0.27pp), desc_has_online (+0.25pp), desc_has_story (+0.34pp),
    has_remote_play (+0.17pp), has_leaderboards (+0.16pp)
